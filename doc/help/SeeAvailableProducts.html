How to see all available database products:

This guide will show you how to easily see which database products are available to create Data Definition Languages (DDL) for.

  1. From the "Options" drop-down menu, select the "Show Database Products Available" menu item.
  2. A popup will be displayed listing which database platforms are available to create a DDL from.

Troubleshooting:

"Show Database Products Available" menu item cannot be selected:

If you are unable to select the "Show Database Products Available" menu item, follow the first steps of our Guide to writing a DDL.
